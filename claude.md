# upnetzero
- เป็น ระบบบันทึกและติดตามการปล่อยก๊าซเรือนกระจก (Carbon Footprint) ขององค์กร สำหรับ มหาวิทยาลัยพะเยา เพื่อมุ่งสู่เป้าหมาย Net Zero / Carbon Neutrality

# Universal Rules
- ส่งโค้ดมาให้รีวิวทุกครั้ง พร้อมบอกตำแหน่งในการแก้ไข และแก้อะไร อย่างไรบ้าง พร้อมเหตุผล
- ห้ามแก้ไขโค้ดโดยไม่มี Unit Test ครอบคลุม
- อ่านไฟล์ที่เกี่ยวข้องก่อนทุกครั้งในการเสนอทางแก้ไข

# Tech Stack
- php
- HeidiSQL
- XAMPP

# Workflow
- *คำสั่งรันหน้าเว็บใน terminal Powershell* C:\xampp\php\php.exe -S localhost:8000

# Negative Constraints
- ห้ามลบไฟล์หรืออะไรโดยไม่ได้อนุญาติ
- ห้ามแก้ไขโค้ดเองโดยไม่ได้รับอนุญาติ
